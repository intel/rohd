InlineSystemVerilog

InlineSystemVerilog mixin Null safety

Allows a Module to define a special type of CustomSystemVerilog which can be inlined within other SystemVerilog code.

The inline SystemVerilog will get parentheses wrapped around it and then dropped into other code in the same way a variable name is.
