Constructs a two-input bit index gate for an abitrary custom functional implementation.

The signal will be indexed by index as an output. Module is in-lined as SystemVerilog, it will use originalindex, where target is index's int value When, the original has width '1', index is ignored in the generated SystemVerilog.
