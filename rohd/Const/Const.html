Constructs a Const with the specified value.

If val is a LogicValue, the width is inferred from it. Otherwise, if width is not specified, the default width is 1. If fill is set to true, the value is extended across width (like ' in SystemVerilog).

Implementation

Const(dynamic val, {int? width, bool fill = false})
    : super(
          name: 'const_$val',
          width: val is LogicValue ? val.width : width ?? 1) {
  put(val, fill: fill);
  _unassignable = true;
}
